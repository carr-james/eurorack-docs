Welcome to the documentation for my collection of DIY Eurorack synthesizer modules.

== Available Modules

=== xref:cem3340-vco::index.adoc[CEM3340 VCO]

A voltage-controlled oscillator based on the classic CEM3340 chip. My first module build, cloned from the https://github.com/kassu/kassutronics/tree/master/documentation/VCO%203340[Kassutronics VCO 3340].

ifdef::show-wip-warnings[]
_Status: Tested - Documentation and front panel in progress_
endif::[]

=== xref:quad-vca-mixer::index.adoc[Quad VCA Mixer]

A four-channel voltage-controlled amplifier and mixer module, cloned from https://musicfromouterspace.com/analogsynth_new/DUALVCA/DLLVCA001.html[Ray Wilson's Dual Log/Linear VCA]. I expanded it from two to four VCAs, added LED output indicators, and included a mixer that normalizes the other three VCA outputs to the fourth jack.

_Status: Untested - PCBs fabricated but not assembled_
endif::[]
